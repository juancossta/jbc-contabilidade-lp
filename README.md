# JBC Contabilidade - Landing Page

Projeto de uma Landing Page moderna para o escritório **JBC Contabilidade**, localizado em Queimados - RJ.

##  Tecnologias Utilizadas
- HTML5 & CSS3
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- JavaScript (Interatividade e Scroll Reveal)
- Google Fonts (Inter e Lexend)

##  Funcionalidades
- Layout Responsivo (Mobile First)
- Integração direta com WhatsApp para orçamentos
- Animações de Scroll Reveal para melhor UX
- Formulário de contacto funcional (simulação de envio)

##  Localização
O escritório está situado na Rua Ver. Marinho Hemetério Oliveira, 853 - Queimados, RJ.
